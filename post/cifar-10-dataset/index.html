How To Get Cifar-10 Dataset

December 13, 2018

How to get cifar-10 dataset on Mac terminal

In this post, I will explain how to get cifar-10 dataset using terminal on mac. The cifar-10 dataset consists 10 classes which are airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck. These classes have 6000 images per class. There are 50000 training images and 10000 test images in this dataset.

Let’s see how to get the cifar-10 dataset :

First of all, I guess you have already installed darknet on your computer. If not, you can download it by clicking this link https://pjreddie.com/darknet/install/

Open your terminal

With using vim ~/.bashrc activate darknet. I’m pretty sure the downloading process will work better.

Open directory which name is data.

cd darknet/data
(data) => wget https://pjreddie.com/media/files/cifar.tgz

(You need to install “wget” if you can’t run this command. Installation wget is ‘brew install wget’ After installed, you can run the command.)

(data) => tar xzf cifar.tgz

After that list cifar directory :

(data) => ls cifar

this cifar directory should contain labels.txt - test - train files.

If you want to see what kinds of classes you have,

(data) => cat cifar/label.txt

and you will see the 10 classes.

We need to generate the paths files. These files will hold all the paths to the training and validation data. Let’s do that:

open directory of cifar

(data) => cd cifar

Write down these two commands:

(cifar) => find `pwd`/train-name\*.png > train.list
(cifar) => find `pwd`/test-name\*.png > test.list

then

cd ../..

After generated paths files, we have to give Darknet some metadata about cifar-10 dataset. Using a text editor which I use Sublime Text, open a new file called ‘cfg/cifar.data’ in the ‘cfg/’ directory. In it, you should have something like;

classes=10
train  = data/cifar/train.list
valid  = data/cifar/test.list
labels = data/cifar/labels.txt
backup = backup/
top=2

Also, we need to make another file called cfg/cifar_small.cfg in the ‘cfg’ directory for making a network to train dataset.

[net]
batch=128
subdivisions=1
height=28
width=28

[convolutional]
batch_normalize=1

[maxpool]
size=2

[convolutional]

[convolutional]

[maxpool]
size=2
stride=2

[convolutional]
batch_normalize=1
filters=32

pad=1
activation=leaky

[convolutional]

filters=128

[convolutional]

[convolutional]

[avgpool]

[softmax]

This one is small network and it is only see how to train a dataset.

Let’s train the cifar-10 dataset :

(darknet) => ./darknet classifier train cfg/cifar.data cfg/cifar_small.cfg
  • Make sure that you correctly wrote your paths files. I got some trouble about it. For example, I wrote
  • * something different like cfg:cfg/cifar.data… ***

After this step, download an image which should depends on dataset as frog,fox or airplane.

then run this code below:

    (darknet) => ./darknet classify cfg/cifar_small.cfg backup/cifar_small_5000.weights ~/location-of-your-image
