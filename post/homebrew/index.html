Homebrew

November 8, 2018

On this blog, I will give brief information about Homebrew and list some basic commands of Homebrew. What is Homebrew? Homebrew is a software package management system that simplifies the installation of software on Mac OS. What does a package manager do? Packages are installable bundles of source code. A package manager is a tool which automates the process of installing, updating, and removing packages. More Information about Homebrew ; Homebrew is written in the Ruby programming language and targets the version of Ruby that comes installed with the macOS operating system. Homebrew installs the stuff you need that Apple didn’t. Homebrew is fast, free and open-source. Doesn’t use ‘sudo’ to install packages. Why would you need a package manager? If you usually use open source software that needs compiling, updating or configuring etc. manually, you would need a package manager and Homebrew is one such package manager. How to Install and Use Homebrew Installation Homebrew : /usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/ Homebrew/install/master/install)”

Paste the above command at a Terminal prompt.

Homebrew Cheatsheet $ brew install packagename : Install a package $ brew uninstall packagename : Uninstall a package $ brew search : This will show everything that Homebrew has. $ brew search packagename : Search a package $ brew list : List installed $ brew list - -version packagename : See what versions you have $ brew info packagename : List versions and information about a package $ brew update : Update all packages you installed. $ brew upgrade : Upgrade all packages you installed. $ brew - -version : See what version of Homebrew you have $ brew leaves: show installed formulae that are not dependencies of another installed formula. $ brew cleanup packagename : Remove old versions Troubleshooting: $ brew config $ brew doctor : fix all the warnings. If you have any trouble, run “brew update” twice and “brew doctor” before creating an issue!
